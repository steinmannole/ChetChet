## Client

**INSTALL:**

    $ cd client
    $ npm install
    ( $ npm install axios@1.3.5 react@18.2.0 react-dom@18.2.0 react-script@5.0.1 stream-chat@8.5.0 stream-chat-react@10.7.5 universal-cookie@4.0.4 web-vitals@2.1.4 )

**START:**

    $ cd client
    $ npm start
  
**BUILD**

    $ npm run build
    $ serve -l 80 -s build

## Server
